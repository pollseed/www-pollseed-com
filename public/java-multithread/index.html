まとめ

  • クラス変数はマルチスレッド間で値が共存されるので、それを期待しないのであればクラス変数を避ける
  • インスタンス変数、クラス変数ともにサーブレットで使う時は設定を見直すか、そもそも共存されても問題ないのかを確認しましょう。サーブレットの場合通常は、1インスタンスに対してマルチスレッドでアクセスをします
