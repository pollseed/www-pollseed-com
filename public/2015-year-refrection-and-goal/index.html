技術バージョン移行メモ

Javaを早く8に移行したい。9もリリース間近なので。
Node.jsは5まで頑張って移行したが、そもそものES5をES6に移行したい。
CentOSはしばらく7に変える気はない。

  • Java 1.6.xx → 1.7.xx
  • Node.js 0.12 → 5.xx
  • CentOS 5.xx → 6.xx
  • Ubuntu 12.xx → 14.xx
